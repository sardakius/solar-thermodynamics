# Solar Thermodynamics

This is a simulation of the Sun that uses the Eddington Solar Model. The full documentation has been uploaded in [video form]() to YouTube. 

The profiles can be founded in the profiles folder in the data folder. To see the pygame visulation, run the `data_reader.py` file. The default option displays the temperature gradient, although the following arguments can be passed to view other properties.

* `-t`, `--temperature `: Temperature
* `-d`, `--density`: Density
* `-p`, `--pressure`: Pressure
* `-e`, `--egr`: Energy Generation Rate
* `-m`, `--mass`: Mass
* `-l`, `--luminosity`: Luminosity
